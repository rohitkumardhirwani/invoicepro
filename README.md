# InvoicePro

InvoicePro is a simple and modern invoice generator built for businesses, freelancers, and service providers. It helps users create professional invoices for both products and services without wasting time.

---

## Features

* Generate invoices quickly
* Add products and services in one invoice
* Automatic subtotal and total calculation
* Currency selected once and applied everywhere
* Clean and professional invoice layout
* Easy to manage customer details
* Responsive and user-friendly interface
* Download or print invoices

---

## Problem Solved

Most invoice systems force users to select or type currency again and again for every product or service.

InvoicePro solves this by:

* Selecting currency only one time
* Automatically applying the same currency to all invoice items
* Saving time and reducing mistakes

---

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* UI/UX: Responsive Design
---

## How It Works

1. Enter customer information
2. Select currency once
3. Add products or services
4. System calculates totals automatically
5. Generate, print, or download invoice

---

## Future Improvements

Open the project in your browser and start generating invoices.

## Project Goal

InvoicePro is designed to make invoice creation faster, cleaner, and easier for everyone.
